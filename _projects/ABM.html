1. Overview

Agent-Based Modeling (ABM) is a powerful computational approach where individual entities (agents) interact within a dynamic environment. These interactions collectively produce emergent behaviour that cannot be reproduced using traditional equation-based modeling methods.

My work in ABM focuses on combining real-world human mobility traces, ML-driven behavioural clustering, and agent stochastic simulation into a unified ABM system to model airborne (COVID-19) and vector-borne (dengue) diseases with high behavioural realism.

Graphical project abstract explaining different layers that coexist within the designed ABM system.

2. Key Features of the Project

  • Real-world GPS mobility modeling from 100 human participants from different occupations.
  • Behavioural extraction using DBSCAN and Spectral Clustering.
  • High-fidelity simulation of COVID-19 and dengue.
  • Transportation-aware exposure modeling (buses, shared mobility).
  • Policy evaluation: quarantine, vaccination, social distancing, hotspot elimination.
  • Hybrid ABM + SEIR framework for epidemiological accuracy.

3. Technical Highlights

  • State-machine models for occupational daily routines.
  • Graph-theoretic contact modeling for exposure estimation.
  • Patch-based vector population modeling for dengue (temperature, humidity, rainfall).
  • Probabilistic transmission for airborne and vector-borne infection.
  • AI-based identification of mobility clusters and risk zones.
